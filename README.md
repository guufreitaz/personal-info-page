📄 Dynamic Personal Info Page

🚀 Technologies
HTML5
CSS3
JavaScript (ES6+)
Fetch API
JSON
⚙️ Features
Data consumption using Fetch API
JSON file reading
Dynamic content rendering on the page
Simple and organized structure
📚 Learning Outcomes
Asynchronous requests (Promises)
DOM manipulation
Separation between data and UI
